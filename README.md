# ansible-provision-ubuntu-desktop


## Setup and run

```bash
sudo apt install -y git
cd ~
git clone https://github.com/bboykov/configure-ubuntu-desktop.git
cd configure-ubuntu-desktop
bash configure_desktop.sh
```
